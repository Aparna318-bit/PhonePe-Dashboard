# PhonePe-Dashboard
Recommend Structre and Order

1. Project Title/Headline

An interactive Power BI Phonepe- dashboard designed to analyze PhonePe payment transactions, customer behavior, and business performance through dynamic visualizations and KPIs.

2. Short Description/Purpose-

* Monitor payment performance and transaction trends.
* Analyze user behavior and service usage.
* Support data-driven business decisions.

3. Tech Stack-

* Power BI
* Power Query
* DAX
* KPI
* Microsoft Excel / CSV

4. Data Source

Sample PhonePe transaction dataset (Excel/CSV)-
Includes transaction details, payment status, users, services, and age segments.
Created for learning and portfolio purposes.

5. Features / Highlights

* KPI Cards (Total Transactions, Transaction Value, Users, Success Rate)
* Monthly Transaction Trend Analysis
* Age Segment Analysis with Drill-through
* Service-wise Transaction Value Analysis
* Top 5 Users by Transaction Value
* Weekday vs Weekend Usage Comparison
* Interactive Filters (Month & Payment Status)
* Business Insights for better decision-making
